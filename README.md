## App To Scan QR Code 

https://user-images.githubusercontent.com/89989829/152540271-423c7ec6-ee53-4533-ae41-b1e71e999e9c.mp4

<br>

**Features:** 
- Scan QR Codes
- Scan 2D Barcodes
- Flash Control while Scanning
- View Scan History
- Open URL links after Scanning
- Can Scan upto a Maximum Distance of 1 metre
- Permission Handling

**Tools Used:** 
- VSCode 
- Flutter 

**Packages Used:** 
- barcode_scan2
- url_launcher

**Features:** 
- Scan QR Codes
- Scan 2D Barcodes
- Scan QR code of the popular Netflix series "Dark" and display character details
- Flash Control while Scanning
- View Scan History
- Open URL links after Scanning
- Can Scan upto a Maximum Distance of 1 metre
- Permission Handling

**Tools Used:** 
- VSCode 
- Dart
- Flutter 

**Packages Used:** 
- barcode_scan2
- url_launcher
- http
- firebase_auth
- cloud_firestore
